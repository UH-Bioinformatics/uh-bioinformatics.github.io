Custom Analyses

The Bioinformatics Core can provide support for the development of custom applications or pipelines in various programming or scripting languages such C++, Perl, Python and R. The core can also, provide support for porting existing bioinformatics applications to run the UH-ITS HPC or for reviewing and optimizing legacy code.

Infrastructure Support

Long or short term data storage: Users supported by COBRE of INBRE can request dedicated, replicated storage for short or long term archival of their data. Other users should consider the Value Storage service provided by Cyber Infrastructure

Large Memory Nodes: The bioinformatics core can provide users from supported grants with dedicated, large memory (512GB RAM) servers. These servers are configured with up-to-date versions of the most popular Bioinformatics tools and applications. For access to the University of Hawaii High Performace Computing cluster, please contact Cyber Infrastructure
